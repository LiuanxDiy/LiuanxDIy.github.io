用51单片机点亮一个 LED

准备工作

  1. 安装 keil5

    点击链接进入 keil 官网，因为是51单片机所以下载 C51 版本。

  2. 安装 stc-isp

    进入官网在软件下载中自行下载最新版即可。

新建一个 keil5 C Project 文件

点击 Project -> new uVision Project ->命名为“点亮一个 LED”。

如图所示：

在器件选择中选择 AT89C52 器件。

在弹出的 Copy’STARTUP.A51’to Project Folder and Add File to Project 窗口中选择否。

在新出的窗口中邮件 Source Group 1 中右键选中图中高亮部分。

新建 .c 程序并命名为 main。

在 main.c 中键入一下代码测试运行。
